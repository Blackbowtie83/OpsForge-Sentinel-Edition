OpsForge is the most advanced open-source digital command post ever built for professional military education (PME), CGSC, War College, Battle Staff, and NCO courses. Built by an MIT PhD Software Engineer & former Apple Application Development Director.
🚀 What is OpsForge?
OpsForge is a complete, interactive military operations center that runs entirely in your browser. It replicates a real Tactical Operations Center (TOC) with native NATO APP-6 symbology (fully compatible with MIL-STD-2525E).

Live NATO APP-6 / MIL-STD-2525E Tactical Map (powered by spatialillusions/milsymbol logic — the gold standard)
Full MDMP (Military Decision Making Process) with step-by-step CGSC/War College fidelity
AI-powered full staff collaboration (S1–S9, G1–G9, Engineer, SPO, SJA, Surgeon, CSM — all officer + NCO bots)
Automatic OPORD + 7 Annexes + Appendices + Overlays generation
Cross-echelon conversion (upload any document from Platoon to Corps → instantly rebuilt at your selected echelon)
Commander’s Placemat with live running estimates
Doctrine Q&A engine (FM 5-0, FM 6-0, FM 1-02.2, ADP 3-90, etc.)
Realistic staff chat — bots communicate exactly like a professional command staff

✨ Key Features

NATO APP-6 Symbol Engine — Full official NATO APP-6 / MIL-STD-2525E symbology with frames, amplifiers, task force, echelon markers, and control measures
Interactive Tactical Map — Leaflet + MGRS + grid lines + drag-and-drop NATO APP-6 symbols + PDF export
Full Staff Bots — 24+ AI bots covering every staff section with real CGSC/War College/Battle Staff/CSM Course expertise
MDMP Automation — Receipt of Mission → COA Development → Wargaming → Orders Production in real time
OPORD Generator — One-click complete orders with all annexes and NATO APP-6 overlays
Cross-Echelon Intelligence — Upload any order/document → bots convert it to your exact echelon
Commander’s Placemat — Live, printable TOC placemat that updates from staff chat
Doctrine AI — Ask anything about FM 5-0, FM 6-0, MIL-STD-2525E, or NATO APP-6 — get doctrinally precise answers
Zero Installation — Single HTML file — works offline after first load

🛠️ Tech Stack

Pure vanilla HTML5 + CSS3 + JavaScript (no frameworks, maximum performance)
Leaflet.js for the tactical map
spatialillusions/milsymbol symbol rendering engine (official NATO APP-6 / MIL-STD-2525E logic)
jsPDF + html2canvas for professional PDF overlays and placemats
Responsive, military-green terminal aesthetic with scanline effects
